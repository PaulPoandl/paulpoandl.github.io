# UniversityFlow – Webshop_Projekt

Dieses Repository enthält den Webshop **UniversityFlow**, der von mir im Rahmen der Lehrveranstaltung *Daten- und Informatikrecht* an der **TU Wien** entwickelt wurde.
UniversityFlow ist ein fiktiver Webshop für eine cloudbasierte SaaS-Anwendung, die den Studienalltag durch Echtzeit-Synchronisation, Campus-Chats und KI-Assistenz produktiver gestaltet. 

Die vollständig funktionale Website ist über GitHub Pages online abrufbar:
https://paulpoandl.github.io
